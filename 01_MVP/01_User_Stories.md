# 01. User Stories: MemoryChain MVP

## 1. Administrator

As an **administrator**, I want to...

*   ...log in to the application securely so that I can manage the organization's assets.
*   ...upload a digital asset (document or image) to the platform so that it can be archived.
*   ...add metadata to the asset (title, author, date, description) so that it can be easily identified and searched.
*   ...see a confirmation that the asset has been successfully uploaded to IPFS and Filecoin so that I know it is securely stored.
*   ...receive a verifiable credential for the asset so that I can share it with others.
*   ...see a list of all the assets that have been uploaded to the platform so that I can manage them.
*   ...customize the application with my organization's name and logo so that it matches our branding.

## 2. Public User

As a **public user**, I want to...

*   ...access a public verification page so that I can verify the authenticity of an asset.
*   ...scan a QR code or enter a unique ID to retrieve information about an asset so that I can be sure it is authentic.
*   ...see the asset's metadata and a link to the asset on IPFS so that I can view it.
*   ...see a confirmation that the asset's credential is valid so that I can trust its authenticity.
